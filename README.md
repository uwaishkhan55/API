# NODEJS EXPRESS MONGODB RESTful API.

## Setup Project

### Step 1
Type in the route directory of this repo in a terminal `npm i` OR `npm install` to install dependencies

### Step 2
Type in a terminal `npm start` to start the server

### Routes
1.retrieve/:id(GET)

2.update(PUT)

3.delete(Post)

4.create(POST)
